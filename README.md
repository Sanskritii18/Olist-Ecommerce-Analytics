# End-to-End Analytics Workflow

This project demonstrates the complete analytics lifecycle using the Brazilian Olist E-commerce dataset.

### Phase 1: Data Cleaning (Microsoft Excel)

- Inspected raw datasets
- Checked for missing values
- Removed duplicate records where applicable
- Standardized column formats
- Validated date and numeric fields
- Performed initial data quality checks

### Phase 2: Data Audit (SQL)

- Explored relational database structure
- Verified primary and foreign key relationships
- Performed data quality checks
- Analyzed customer, seller, product, payment, and order data
- Executed analytical SQL queries to understand business trends
- Validated dataset consistency before visualization

### Phase 3: Business Intelligence (Power BI)

- Imported cleaned data using Power Query
- Built an optimized Star Schema
- Developed DAX measures
- Designed interactive dashboards
- Generated business insights and recommendations
